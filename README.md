# Software-Engineering-Project-Binus
ThrashIn ♻️: Platform pengelolaan sampah cerdas dengan layanan jemput berlangganan, penjadwalan otomatis, dan gamifikasi leaderboard. Pantau progres daur ulang, raih badge, serta belanja produk ramah lingkungan via katalog terintegrasi. Dilengkapi peta lokasi titik buang sampah untuk gaya hidup lebih hijau! 🌍✨


USER STORY 1: Layanan Jemput Sampah Berlangganan
Sebagai pemilik rumah, saya ingin berlangganan layanan jemput sampah dengan jadwal yang bisa saya atur sendiri, baik harian maupun mingguan, supaya sampah di rumah tidak menumpuk dan saya tidak perlu repot membuangnya.

Penjelasan Fitur

User dapat memilih paket langganan mingguan atau bulanan.User dapat menentukan jadwal penjemputan, misalnya setiap hari atau hari tertentu seperti Senin dan Kamis.Jadwal dapat diubah maksimal satu hari sebelum waktu penjemputan.Jika user sedang bepergian atau liburan, tersedia fitur untuk menjeda langganan sementara dan mengaktifkannya kembali nanti.Sampah harus dipisahkan menjadi organik dan anorganik sesuai ketentuan.Setelah pembayaran berhasil, sistem otomatis membuat jadwal penjemputan.

Acceptance Criteria

Story dianggap selesai jika:

- **User dapat memilih paket langganan mingguan atau bulanan.**
- **User dapat memilih dan mengatur hari penjemputan.**
- **User dapat mengubah jadwal maksimal H-1 sebelum penjemputan.**
- **Tersedia tombol untuk menjeda dan mengaktifkan kembali layanan.**
- **Sistem otomatis membuat jadwal setelah pembayaran berhasil.**
- **User mendapatkan notifikasi sebelum hari penjemputan.**
- **Tersedia riwayat penjemputan di aplikasi.**

Tambahan yang diperlukan

Sistem menyediakan beberapa metode pembayaran.
Ada notifikasi pengingat sebelum jadwal jemput.
User dapat memberikan rating atau ulasan kepada petugas penjemput.

USER STORY 2: Gamifikasi dan Leaderboard

Sebagai pengguna yang kompetitif, saya ingin mendapatkan poin dan badge dari sampah daur ulang yang saya setor serta melihat peringkat saya di leaderboard, agar saya lebih termotivasi dan berkesempatan mendapatkan hadiah.

Penjelasan Fitur

Setiap satu kilogram sampah daur ulang yang valid akan mendapatkan 10 poin.Petugas akan memverifikasi berat dan jenis sampah sebelum poin dimasukkan ke sistem.User dapat memperoleh badge berdasarkan jumlah poin tertentu.Leaderboard menampilkan peringkat berdasarkan poin tertinggi.Leaderboard akan di-reset setiap tanggal 1 setiap bulan.Tiga user dengan poin tertinggi di akhir bulan akan mendapatkan hadiah.

Acceptance Criteria

Story dianggap selesai jika:

- Poin bertambah otomatis setelah diverifikasi petugas.
- *Leaderboard menampilkan peringkat user secara real-time.*
- User mendapatkan notifikasi saat memperoleh badge.
- *Sistem mereset ranking setiap awal bulan.*
- *Riwayat pemenang tersimpan dalam sistem.*
- *Ada proses verifikasi untuk mencegah kecurangan.*

Tambahan yang Disarankan

*Tersedia progress menuju badge berikutnya.
Ada sistem level berdasarkan total poin.*
User dapat membagikan pencapaian ke media sosial.
Hadiah bisa berupa voucher atau diskon langganan.

USER STORY 3: Peta Lokasi dan Produk Daur Ulang

Sebagai pengguna yang sedang berada di luar rumah, saya ingin melihat lokasi tempat sampah terdekat dan membeli produk hasil daur ulang, agar saya tetap bisa peduli lingkungan dan mendukung produk ramah lingkungan.

Penjelasan Fitur

Fitur peta menampilkan lokasi user saat ini dan lokasi tempat sampah terdekat.User dapat menekan pin lokasi untuk melihat alamat dan detail lokasi.Tersedia tombol untuk navigasi menuju lokasi tersebut.Fitur produk daur ulang menampilkan katalog produk dengan foto, deskripsi, dan harga.User dapat melakukan pemesanan langsung melalui aplikasi.User mendapatkan notifikasi terkait status pesanan.

Acceptance Criteria

Story dianggap selesai jika:

- Peta menampilkan lokasi user dan pin tempat sampah.
- Pin lokasi dapat ditekan dan menampilkan detail.
- Tersedia tombol navigasi menuju lokasi.
- *Halaman produk menampilkan foto, deskripsi, dan harga.*
- *User dapat melakukan checkout dan pembayaran.*
- User menerima notifikasi status pesanan.

Tambahan yang Disarankan

Tersedia filter produk berdasarkan kategori atau harga.
User dapat memberikan ulasan produk.
Tersedia estimasi pengiriman.
User dapat menyimpan produk ke daftar favorit.
